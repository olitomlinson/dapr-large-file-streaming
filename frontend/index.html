Architecture

Browser → frontend:8080 (static)
        ↓ (API call)
Browser → nginx:80 → nginx-dapr:3500 → file-generator-dapr:3500 → file-generator:8003

The entire API chain uses chunked transfer-encoding without buffering.

Note: file-generator-dapr buffers the entire response (~300% overhead). See STREAMING_RESPONSE_ISSUE.md for details.
